# DreamOn

An LSPosed module that enables P10 exclusive screensavers for Pixel 6+ devices.

## Description

For some reason Google restricts the following new screen saver options to Pixel 10 series:
* Everyday Clock
* Home Controls
* Pilot Bold Clock
* Weather
This module is here to fix this!

## Installation

1. Install the APK.
2. Enable the **DreamOn** module in **LSPosed Manager**.
3. Ensure **Dreams** (`com.google.assistant.hubui`) and **Pixel Weather** (`com.google.android.apps.weather`) are selected in the module's scope.
4. Restart the **Dreams** and **Pixel Weather** apps.
5. Navigate to *Settings -> Display & Touch -> Screen saver* and choose your new screen saver!

## Compatibility

* **Devices**: Pixel 6 series and newer.
* **OS**: Tested on Android 16 QPR3.
